КАДАСТР ОБЪЕКТОВ НЕДВИЖИМОСТИ
===============================

Программа для ведения единого реестра адресов объектов недвижимости.
Позволяет добавлять адреса в общую базу, проверять дубликаты, загружать
списки из Google Таблиц и отслеживать вклад каждого сотрудника.

ОСНОВНЫЕ ВОЗМОЖНОСТИ
--------------------

  - Проверка адреса перед добавлением (исключает повторы)
  - Ручной ввод объектов: адрес, название, ссылка
  - Массовая загрузка из Google Таблиц (CSV-файл)
  - Автоматическая привязка объектов к сотрудникам из таблицы
  - Произвольные дополнительные поля (кадастровый номер, площадь и т.д.)
  - Выгрузка полного реестра в Excel (CSV)
  - Рейтинг сотрудников по количеству внесённых объектов
  - Личный кабинет с историей добавлений
  - Администратор может управлять пользователями и записями
  - Защита паролем и ограничение попыток входа


ЗАПУСК
------

  1. Дважды щёлкните файл «ЗАПУСК.bat»
  2. Дождитесь сообщения «Приложение запущено»
  3. Откроется браузер с программой

  Для остановки — закройте окно запуска или дважды щёлкните
  файл «ОСТАНОВКА.bat».


ПЕРВЫЙ ВХОД
-----------

  Администратор создаётся автоматически при первом запуске.

  Логин:   admin@cadastre.ru
  Пароль:  Admin123

  После входа вы увидите вкладку «Админ» в верхнем меню —
  через неё можно управлять пользователями и объектами.


СИСТЕМНЫЕ ТРЕБОВАНИЯ
--------------------

  - Windows 10/11
  - 500 МБ свободного места на диске
  - Доступ в интернет (для загрузки библиотек при первом запуске)
